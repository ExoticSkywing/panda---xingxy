# 卡密编辑功能

## 功能描述

为卡密管理添加编辑功能，支持修改已导入的卡密信息。

## 新增功能

| 功能 | 入口 | 说明 |
|------|------|------|
| **单条编辑** | 列表页备注列「编辑」按钮 | 可修改卡号、密码、备注 |
| **批量修改** | 批量操作下拉菜单 | 选中多条后统一修改备注 |

## 文件变更

| 文件 | 操作 |
|------|------|
| `xingxy/inc/card-edit.php` | 新增 |
| `xingxy/init.php` | 引入新模块 |

## 使用说明

1. **单条编辑**：点击备注列的「编辑」按钮 → 弹窗修改 → 保存
2. **批量修改**：勾选多条卡密 → 批量操作选择「批量修改备注」→ 应用 → 输入新备注 → 保存
